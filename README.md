# Journal Apps with NestJs

## Description
A simple CRUD Rest API app that i have been build with NestJs, TypeORM and MongoDB.

## Installation
1. Clone this project with `git clone #repo-name`
2. 
```bash
$ npm install
```
or
```bash
$ yarn install
```

## Running the app

```bash
# development with npm
$ npm run start or

# watch mode
$ npm run start:dev
# watch mode with yarn
$ yarn start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
